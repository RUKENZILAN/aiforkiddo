# AI Learning Guide for Kids — README

A curated, single-page HTML guide that helps children, parents, and teachers discover trusted artificial-intelligence learning resources. The page links only to official sources and does not host any videos or copyrighted content itself.
The page is published here: https://aikiddo.org/
---

## What is this?

This project is a hand-picked collection of AI learning resources for kids. It was built with input from both an educator and an IT specialist, so each recommendation is checked for age-appropriateness, licensing, and privacy.

The guide is delivered as one self-contained HTML file: `index.html`.

---

## Features

- **Bilingual:** Switch between Turkish (`TR`) and English (`EN`) with one click. The selected language is remembered for the next visit.
- **Age filtering:** Filter platforms and curricula by age group — Ages 5–7, Ages 8–12, Ages 13+, or For teachers.
- **Curated cards:** Each card shows the resource name, age range, type, cost badge, a short description, and a direct link to the official site.
- **Video channels:** A separate section lists recommended YouTube channels for learning about AI, coding, science, and math.
- **No content hosting:** All videos and materials stay on their original provider sites. This page only links to them.
- **Responsive design:** Works on desktop, tablet, and mobile browsers.

## How to use

1. Open `index.html` in any modern web browser.
2. Use the **TR / EN** buttons at the top-right to choose your language.
3. Use the age-filter buttons under **Platforms & Curricula** to narrow resources by age group.
4. Click any **Go to platform →** or **YouTube** link to visit the official resource.

No internet connection is required to view the page itself, but the linked platforms and videos require an internet connection.

## Important notes

- **No content is copied or re-published.** All links point to the original provider's official website or YouTube channel.
- **Age badges are suggestions, not strict rules.** Adjust choices based on the child's interest and maturity.
- **Adult guidance is recommended**, especially for children under 13 creating accounts on external platforms.
- **Free does not mean copyright-free.** The "Free" badge only means there is no access cost.
